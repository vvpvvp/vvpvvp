# vvpvvp
在给定的范围内返回一个随意值。

## Install

```sh
npm install vvpvvp
npm install --save vvpvvp
```

### Example

```javascript
let a = vvpvvp([1, 2], 2);
//1.48
let a = vvpvvp([3], 3);
//3.123
let a = vvpvvp([-1, 3], 3);
//-0.123
```

## API

### params

vvpvvp([start, <end>], <toFixed = 2>)
